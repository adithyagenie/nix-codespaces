# nix-codespaces

Minimal devcontainer configured with Nix.

Start your Github Codespace from the master branch of this repo.

You can switch the image in `.devcontainer/devcontainer.json` to `"mcr.microsoft.com/devcontainers/base:ubuntu"` if you do not want a huge image with all the languages and tools installed by default.

devcontainer source: [nixpkgs](https://github.com/NixOS/nixpkgs/blob/master/.devcontainer/devcontainer.json)
